## FAITS RETENUS PAR THÈME

### Histoire
- Traité du 1er juin 1784 : France → Suède, échange contre entrepôt à Göteborg ✓
- 7 septembre 1785 : déclaration port franc par décret royal suédois ✓
- 1787 : rebaptisée Gustavia en hommage à Gustave III ✓
- Apogée début XIXe siècle : ~3 900 habitants à Gustavia ✓
- 10 août 1877 : traité de rétrocession, 320 000 francs payés par la population ✓
- 1878 : retour officiel à la France, port franc maintenu ✓

### Port
- 300 places, dont ~30 visite à quai, ~200 mouillages ✓
- Profondeur 8 m avant-port ✓
- Longueur max : 60 m ✓
- Amarrage méditerranéen (ancre + poupe à quai) ✓
- Bucket Regatta (mars) + Voiles de Saint-Barth (avril) ✓

### Shopping
- 3 artères : Quai de la République, Rue du Général de Gaulle, Rue du Roi Oscar II ✓
- Zone franche : pas de TVA ✓
- Maisons : Cartier, Bulgari, Chopard, LV, Prada, Hermès, D&G, Ralph Lauren ✓
- Locales : Poupette St Barth, St Barth French West Indies (1989), Clic Gallery ✓
- Horaires : fermeture midi-15h, mercredi AM fermé ✓

### Gastronomie / Nightlife
- Le Select : fondé 1949, Marius & Helen Stakelborough, plus vieux bar de l'île ✓
- Le Select : Jimmy Buffett → Cheeseburger in Paradise (années 1970) ✓
- Bonito : ouvert 2009, chef-propriétaire Laurent Cantineaux ✓
- Bagatelle : dîner → DJ vers 22h-23h ✓
- Bar de l'Oubli : apéritif coucher de soleil ✓
- Ociela / Barry Rooftop : vue panoramique sur port ✓

### Culture
- Wall House : 2 rue de Pitea, Pointe de Gustavia ✓
- Wall House construit époque suédoise, estimé 1791-1801 (date exacte inconnue) ✓
- Restauré en 1995 par les Compagnons ✓
- Collection permanente : photos, objets, costumes d'époque ✓
- Entrée gratuite ✓
- Horaires : Lun/Mar/Jeu/Ven 9h30-12h30 et 15h-19h, Mer 14h-19h, Sam 9h30-13h [À VÉRIFIER]

### Forts
- 3 forts suédois : Fort Gustav, Fort Karl, Fort Oscar ✓
- Fort Karl : ouest de Gustavia, ~34 m altitude ✓
- Dédié au duc Karl de Sudermanie (futur Karl XIII), frère de Gustave III ✓
- Géré par le Conservatoire du littoral ✓

### Shell Beach
- Anse de Shell Beach = Plage des Grands Galets ✓
- 10 min à pied du port ✓
- Millions de coquillages polis (pas de sable ordinaire), teinte rosée ✓
- Vue sur Saba et Saint-Eustache ✓
- Eau calme, idéale pour nager ✓
